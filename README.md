# ImgCropping

一个纯前端（React + Vite + TypeScript）的图片裁切工具。

## 功能

- 导入方式：
  - 点击上传图片
  - 拖拽图片到页面
  - 直接从剪贴板粘贴图片（`Ctrl/Cmd + V`）
- 自动预处理：
  - 输入图片宽度统一缩放到 `1920`
- 三种裁切模式（输出固定 `1920 x 1080`）：
  - 顶部裁切：`0-1080`
  - 任意起始：`n-(n+1080)`
  - 底部裁切：底部向上 `1080`
- 任意起始支持两种输入：
  - 直接输入像素 `n`
  - 输入页码自动换算（第 1 页从 `0px` 开始，公式 `(页码-1)*1080`）
- 越界处理：
  - `n` 允许越界，超出图像区域自动透明补白
- 结果操作：
  - 复制图片到系统剪贴板（PNG）
  - 下载 PNG

## 技术栈

- React 18
- TypeScript
- Vite 5
- Canvas 2D（前端本地处理，无后端）

## 本地运行

```bash
pnpm install
pnpm run dev
```

打开浏览器访问终端输出地址（默认通常是 `http://localhost:5173`）。

## 生产构建

```bash
pnpm run build
pnpm run preview
```

## 使用说明

1. 导入图片（上传/拖拽/粘贴任一方式）。
2. 选择裁切模式：
   - 顶部 `0-1080`
   - 任意起始 `n-(n+1080)`
   - 底部向上 `1080`
3. 如果是“任意起始”：
   - 可输入像素 `n`
   - 或输入页码自动换算到像素
4. 查看预览后，点击“复制图片”或“下载 PNG”。

## 兼容与权限说明

- 图片复制依赖浏览器 Clipboard API：
  - 需要在安全上下文（HTTPS 或 localhost）
  - 若浏览器不支持或权限被拒绝，可使用“下载 PNG”
- 透明补白在查看器中可能显示为棋盘格背景（页面已内置棋盘格预览背景）
